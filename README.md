# Customer Shopping Behavior Analysis

## Project Overview
This project focuses on analyzing customer shopping behavior to identify purchasing patterns, customer segments, and key factors influencing buying decisions. The goal is to convert raw customer data into actionable business insights that can support marketing strategy, customer engagement, and sales optimization.

The project follows an end-to-end data analytics workflow similar to real-world industry use cases.

---

## Objectives
- Analyze customer purchasing patterns and behavior trends
- Identify high-value customers and repeat purchase behavior
- Understand the impact of discounts, demographics, and seasonality
- Support data-driven business decision-making

---

## Tools & Technologies
- Python (Pandas, NumPy, Matplotlib, Seaborn)
- SQL
- Power BI
- Jupyter Notebook

---

## Project Workflow
1. Data cleaning and preprocessing using Python  
2. Exploratory Data Analysis (EDA)  
3. SQL-based analysis for customer segmentation and loyalty insights  
4. Interactive dashboard creation using Power BI  
5. Business insights and recommendations documentation  

---
